## Manuels de formation (hébergés publiquement par des centres de formation)

3. **Amadeus Quick Reference Guide — cryptic entries**
   https://usermanual.wiki/Pdf/AmadeusQuickReferenceGuide.2132884153.pdf
   → LE plus utile au quotidien : toutes les entrées cryptiques par thème (sign-in, AN/SN/TN,
   SS, NM/AP/RF, tarification FX*, TTP, queues…). Référence pour missions 03 (messages) et 02+.
4. **Amadeus Training Manual — Vista/Egypt (basic course)**
   https://www.air.flyingway.com/books/amadeus/vista-amadeus-egypt.pdf
   → Cours complet débutant : séquences pédagogiques réalistes — utile pour la Phase 6
   (exercices guidés) : nos exercices suivront cette progression type.
5. **The Complete Amadeus Manual (Jasir Alavi)**
   https://air.flyingway.com/books/amadeus/Amadeus_Guide.pdf
   → Manuel généraliste détaillé, bon complément croisé quand 3 et 4 divergent.

## Comment s'en servir dans les missions

- Un doute sur un message d'erreur → chercher dans 3 puis 5 ; consigner la réponse dans
  `docs/ERREURS-AMADEUS.md` avec la référence (doc + page).
- Un doute sur un format d'entrée/sortie → 3 (format) puis 1 (comportement écran).
- Construction des exercices (Phase 6) → dérouler la progression de 4.
- Si un lien meurt : chercher le titre exact — ces manuels sont largement mirrorés.
